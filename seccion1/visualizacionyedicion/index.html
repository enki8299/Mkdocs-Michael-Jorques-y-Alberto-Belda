Comandos de Visualización y Edición

Estos comandos se utilizan para ver el contenido de los archivos de texto.

Comando Función Ejemplo de Uso
cat Muestra el contenido completo de un archivo (Catenate). cat readme.md
less Muestra el contenido página por página, permitiendo desplazarse hacia adelante y hacia atrás. less archivo_grande.log
head Muestra las primeras 10 líneas de un archivo. head -n 5 archivo.txt (muestra las primeras 5)
tail Muestra las últimas 10 líneas de un archivo. tail -f logs.txt (sigue el archivo en tiempo real)
grep Busca patrones de texto dentro de archivos. grep "error" archivo.log
