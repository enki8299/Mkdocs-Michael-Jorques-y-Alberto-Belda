⚙️ Comandos del Sistema y Procesos

Comandos para interactuar con el sistema operativo, gestionar usuarios y procesos.

Comando Función Ejemplo de Uso
ps Muestra información sobre los procesos que se están ejecutando. ps aux
top Muestra una vista dinámica y en tiempo real de los procesos del sistema y el uso de recursos. top
kill Envía una señal a un proceso para detenerlo (a menudo la señal SIGTERM). kill 1234 (1234 es el ID del proceso)
df Muestra el espacio libre en disco (Disk Free). df -h (formato legible)
du Muestra el uso del disco de un archivo o directorio (Disk Usage). du -sh /home/usuario
whoami Muestra el nombre de usuario efectivo actual. whoami
sudo Ejecuta un comando con privilegios de superusuario (Super User Do). sudo apt update
